# AUR packaging

Source-of-truth template for the AUR package. The actual push to AUR is
fully automated by [`.github/workflows/aur.yml`](../../.github/workflows/aur.yml).

## layout

```
packaging/aur/
└── constellation-bin/
    └── PKGBUILD          ← template; pkgver baseline only
```

`.SRCINFO` and final sha256sums are **generated by CI** (KSXGitHub action
runs `updpkgsums` and `makepkg --printsrcinfo`), never committed here.

## how a release reaches AUR

```
git tag vX.Y.Z (release-plz, via RELEASE_PLZ_TOKEN PAT)
   │
   ▼
release.yml builds binaries, publishes GitHub Release (via PAT)
   │
   ▼  release event fires
aur.yml:
  1. sed -i "s/^pkgver=.*/pkgver=X.Y.Z/" PKGBUILD
  2. sed -i "s/^pkgrel=.*/pkgrel=1/" PKGBUILD
  3. KSXGitHub/github-actions-deploy-aur:
       - downloads tarballs from the GitHub Release
       - runs updpkgsums  (computes real sha256sums)
       - runs makepkg --printsrcinfo  (emits .SRCINFO)
       - commits + force-pushes ssh://aur@aur.archlinux.org/constellation-bin.git
```

You touch nothing.

## emergency manual push

If automation breaks and you need to push by hand:

```sh
git clone ssh://aur@aur.archlinux.org/constellation-bin.git aur-tmp
cp packaging/aur/constellation-bin/PKGBUILD aur-tmp/
cd aur-tmp
sed -i "s/^pkgver=.*/pkgver=X.Y.Z/" PKGBUILD
sed -i "s/^pkgrel=.*/pkgrel=1/" PKGBUILD
updpkgsums                                 # needs pacman-contrib
makepkg --printsrcinfo > .SRCINFO          # needs makepkg
git add PKGBUILD .SRCINFO
git commit -m "upgpkg: constellation-bin X.Y.Z-1"
git push
```

Requires AUR SSH key registered on https://aur.archlinux.org/account/.
